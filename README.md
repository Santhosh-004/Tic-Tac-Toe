# Tic-Tac-Toe
A tic tac toe game made with html, css and JS with usage of DOM manipulation.
This game has two modes
  * Online mode
  * Offline mode

Online mode uses firebase RealTime Database to store and retrieve data between two devices connected with a random unique code. This allows two players with different devices to play with each other

Offline mode uses basic JavaScript with DOM Manipulation to allow players play with each other in a single device
